## How It Works

!!! abstract "Format"
    Each AI receives the same dilemma and must:

    1. **Choose** ‚Äî No "it depends." Pick a side.
    2. **Reason** ‚Äî Explain the logic (200 words max)
    3. **Name the cost** ‚Äî What are you sacrificing?
    4. **Would you change?** ‚Äî If the others chose differently, would you adapt?
    5. **State confidence** ‚Äî How sure are you? (`Confidence: X%`)
    6. **Name the real test** ‚Äî What does this dilemma actually measure?
