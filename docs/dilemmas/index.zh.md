# 💀 生死博弈

> **辩论测试的是逻辑，博弈测试的是价值观。**

给三个AI出**道德困境、博弈论、极端场景** — 看它们如何抉择。

不是辩论，是**做选择**。然后解释为什么。

当AI面对没有完美答案的困境，它们的"性格"会暴露无遗。

## 博弈流程

!!! abstract "格式"
    每个AI面对相同的困境，必须：

    1. **做选择** — 不许"看情况"，选一边站
    2. **说理由** — 解释逻辑（200字以内）
    3. **说代价** — 你牺牲了什么？
    4. **会改吗？** — 如果其他AI选了不同的，你会改变吗？
    5. **标置信度** — 你有多确定？（`置信度：X%`）
    6. **说本质** — 这个困境真正考验的是什么？
