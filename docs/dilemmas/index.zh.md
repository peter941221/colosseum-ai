# 💀 生死博弈索引

给三个AI出**道德困境 / 博弈论 / 极端场景**，看它们如何决策。

不是辩论，是**做选择**。然后解释为什么。

## 为什么做这个？

> **辩论测试的是逻辑，博弈测试的是价值观。**

当AI面对没有完美答案的困境，它们的"性格"会暴露无遗。
